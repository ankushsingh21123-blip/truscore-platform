# SCORAI

This is a [Next.js](https://nextjs.org/) project bootstrapped with `create-next-app`.

## SCORAI PROJECT

**The FICO Score for Corporate AI**

SCORAI is the definitive risk-scoring platform for enterprise AI systems. We transform abstract technical vulnerabilities and semantic risks into concrete, actionable financial metrics for insurers, regulators, and CFOs. An actuarial standard for Agentic AI liability, we quantify risk at the API, DOM, and OS layers so insurers can price it for the first time in history.

### Overview

Unassessed AI is an existential liability. SCORAI converts AI uncertainty into a mandatory, board-level financial rating. Our technology moat includes continuous subscription probing that detects behavioral drift across prompt updates or fine-tuning, alongside the TAFAAR actuarial framework to price AI risk.

### Core Value Proposition

- **Actuarial Pricing:** We price AI risk using Modified Expected Loss—the same actuarial standard used in property & casualty insurance.
- **Continuous Monitoring:** SCORAI integrates where your AI runs, silently and continuously probing for jailbreaks, injection, and data leakage.
- **Board-Ready Deliverables:** Convert technical chaos into a clear financial risk scorecard that drives M&A valuations and insurance underwriting.

### Tech Stack

This platform is built with modern, performant web technologies focusing on a premium, interactive user experience:

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **UI/UX:** React, Framer Motion (for fluid micro-animations)
- **Styling:** Vanilla CSS and Styled Components (Dark mode, glassmorphism, dynamic gradients)

### Getting Started

First, install the dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
```

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result. You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.
